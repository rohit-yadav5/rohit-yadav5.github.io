## Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties, CSS Grid, Flexbox, scroll animations
- **Vanilla JS** — typing animation, scroll fade-in, mobile nav
- **GitHub Pages** — hosting (pushes to `main` deploy automatically)

---

## Local development

No install needed. Just open the files directly in your browser:

```bash
git clone https://github.com/rohit-yadav5/rohit-yadav5.github.io
cd rohit-yadav5.github.io
open index.html
```

Or use a local server for accurate link behaviour:

```bash
# Python
python3 -m http.server 8080

# Node
npx serve .
```

Then open `http://localhost:8080`.

---

## Deployment

Every push to `main` is automatically deployed by GitHub Pages. No CI, no build step.

```bash
git add .
git commit -m "your message"
git push origin main
# live in ~30 seconds
```
